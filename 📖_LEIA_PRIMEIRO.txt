
### Step 2: Setup Ambiente

## 💡 DICAS IMPORTANTES

1. **Mantenha simples na Fase 1**
   - MVP = agents gerando estrutura + scripts
   - Vídeos reais = Fase 2 (ElevenLabs, HeyGen, FFmpeg)

2. **Teste frequentemente**
   - `curl` é seu melhor amigo
   - Logs = debugging super importante

3. **Use recursos da comunidade**
   - Claude docs: https://docs.anthropic.com
   - Supabase docs: https://supabase.com/docs
   - Railway: https://docs.railway.app

4. **Não reinvente a roda**
   - Padrão em `research-forge.ts` = use para todos agents
   - Queue pattern em `queue/index.ts` = escalável

5. **Deploy cedo, often**
   - MVP produção = semana 1
   - Feedback loop = maior velocidade
